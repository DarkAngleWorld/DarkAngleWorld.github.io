git的使用

一.git的安装

  1. 点击此处去下载git环境

  2. 终端编译安装

    1. 现在就基本上已经安装成功了，可以在终端上使用 $ git help 语句去测试下
      注：坑能会有版本更新的问题，加入上方的代码无法执行，请点击这里

二.git的初始化配置

主要就写下用户名和Email就行了

三.git同步到本地

1. 项目已存在，同步项目到本地

. $ cd 目录地址             //第一步//选择下载到本地的地址，加入不写这句，会默认是当前的路径
. $ git clone  http地址    //第二步//下载到本地

2. 项目在本地，同步到云端

. $ cd 项目目录地址             //第一步//选择项目地址
. $ git remote rm origin      //第二步//移除项目本地线程分支
. $ git remote add origin 项目https地址  //第三步//为本地项目添加远程连接
. $ git pull --rebase origin master     //第四步//pull线程分支到本地
. $ git push -u origin master           //第五步//提交本地项目到云端

四.git的简单操作

git提交修改的主要四步骤

git的其他简单操作

. $ git status            //查询项目当前状态，例如当前分支是什么，是否需要提交等
. $ git diff              //查询当前修改东西的详情 
. $ git fecth             //同步远程分支到本地
. $ git branch -a         //查看当前分支列表
. $ git checkout 分支名称  //切换当前分支
