Xcode8 使用插件

  • Xcode8 使用插件

这几天升级了Xcode8发现插件失效了，经过谷歌终于找到了可以重新使用插件的方法

  • 第一步

    $ gem install update_xcode_plugins

    注：假如报错就使用这句命令执行：$ sudo gem install update_xcode_plugins

  • 第二步

    update_xcode_plugins

  • 第三步

    update_xcode_plugins –unsig

    注:这里会给出两个选项，我们选择Xcode(8.0) ，同时也会询问Y/N,选择Y。然后你就需要漫长的等待了，这一步需要的时间比较长，因为他需要 copy Xcode。

  • 最后

    这一步其实是没有必要的，因为已经成功了只是需要在这里提醒下插件如何使用，上面执行完后，你的应用程序会添加出如下图所示的应用，平常使用的时候就是用Xcode-unsigned.app就行了，但是提交版本打包到AppStore时还请使用原版的Xcode.app!
